Technical Specifications

ESP32 Bridge

  • WiFi: 802.11 b/g/n (2.4GHz)
  • Power: 20dBm maximum transmit power
  • UART: 115200 baud, 8N1 configuration
  • Protocol: Custom binary with CRC16 validation
  • Heartbeat: 5-second interval

iOS Application

  • Platform: iOS 14.0+
  • Framework: SwiftUI with Combine
  • Networking: TCP socket communication
  • Storage: Core Data with UserDefaults
  • Threading: Async/await with actors

Communication

  • Protocol: Custom binary over TCP
  • Port: 23 (standard telnet port)
  • Validation: CRC16 polynomial 0x8005
  • State Machine: Robust packet parsing
  • Error Handling: Auto-reconnection

Control Parameters

  • PID: P, I, D for each axis
  • Power: Motor power levels
  • Follow: Speed and filtering
  • RC: Remote control settings
  • Limits: Position constraints
